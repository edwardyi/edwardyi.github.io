Using RegExr

正規表達式

  • .+:某
    • .+searchItem
  • .:代表任何東西
    • 任意一個東西
    • 小丑卡，可以代表任何東西
    • 萬用任意(wildcard)
    • 如果想要match到點的話,要跳脫/.
    • 王…
    • 匹配3.14 => 3.14
  • \w:
    • 所有的數字、文字和底線(可以用來檢查用戶名稱)
    • \W:\w的補集(加減乘除)
  • \d:
    • 匹配數字
    • \D:\d的補集(除了數字以外)
  • \s:
    • 匹配空格、tab
    • \S:\s的補集(除了空白以外)
  • []:可以是甚麼的字元
    • [^b]og:不是b
    • 轉成unicode:19968-40869
  • ():分組(常用)
    • (\w+)
  • *:匹配所有
    • {0,}
  • +:匹配至少一個字元
    • {1,} at least 1
  • ?:匹配零個或1個
    • {0,1}
  • {}:要出現幾次甚麼東西
    • z{3}:zzz
